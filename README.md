## Getting started

1. Install [Node.js](https://nodejs.org/) and npm.
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server with live reload:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the project for production:
```sh
npm run build
```

Preview the production build locally:
```sh
npm run preview
```
